## Pre-Capture Preparation

### Environment Setup
- [ ] Device charged to 100%
- [ ] Connected to stable WiFi
- [ ] Device language set to English
- [ ] Low power mode DISABLED
- [ ] Notifications cleared
- [ ] Date/time appropriate (morning/afternoon)
- [ ] Latest app version installed

### App State Preparation
- [ ] App logged in with test account
- [ ] Test data populated (flavor profiles, items)
- [ ] No empty states or loading indicators
- [ ] Images cached and loaded
- [ ] No error messages visible
- [ ] Fresh app restart completed

## Post-Capture Validation

### File Verification
- [ ] All screenshots are PNG format
- [ ] Resolution matches device specifications exactly
- [ ] No transparency in images
- [ ] File sizes reasonable (< 8MB each)
- [ ] All files named correctly

### Quality Check
- [ ] Images are sharp and in focus
- [ ] No blur or pixelation
- [ ] Consistent appearance across screenshots
- [ ] No visual artifacts or glitches
- [ ] Proper color representation

### Content Review
- [ ] No sensitive user data visible
- [ ] No placeholder content
- [ ] No error messages or loading states
- [ ] No development/debug UI elements
- [ ] Appropriate test data only

### Backup
- [ ] Original screenshots backed up
- [ ] Working copies saved to `/working/raw-captures/`
- [ ] Final versions in appropriate directories
- [ ] README updated with completion status

---

## Submission Preparation

### Apple App Store
- [ ] Login to App Store Connect
- [ ] Navigate to app > Screenshots
- [ ] Upload iPhone 15 Pro Max screenshots (5)
- [ ] Upload iPad Pro 12.9" screenshots (4)
- [ ] Add captions for each screenshot (45 char max)
- [ ] Preview on different device previews
- [ ] Save and verify

### Google Play Store
- [ ] Login to Google Play Console
- [ ] Navigate to Main Store Listing > Screenshots
- [ ] Upload phone screenshots (minimum 2, recommended 5)
- [ ] Upload tablet screenshots (recommended 3)
- [ ] Add descriptions for each
- [ ] Preview on different device previews
- [ ] Save and verify
