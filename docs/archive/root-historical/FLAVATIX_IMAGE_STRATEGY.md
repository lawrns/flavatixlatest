# Flavatix Image Strategy: Complete Implementation Roadmap

## Executive Summary

Flavatix will leverage a hybrid image sourcing strategy combining:
- **Free platforms** (Unsplash, Pexels, Pixabay) for 70% of content
- **Paid subscriptions** (Adobe Stock or Shutterstock) for 20% of hero/landing images
- **User-generated content** (UGC) for 10% of authentic/community features

This approach balances budget constraints with brand consistency and premium positioning.

## Part 2: Landing Page Image Integration

### Landing Page Structure

#### Section 1: Hero Banner
**Purpose**: Immediate brand impression, value proposition

**Image Specs**:
- Dimensions: 1920x1080px
- Format: WebP + JPEG fallback
- File Size: 80-120KB (optimized)
- Overlay: Dark gradient (opacity 0.4-0.5)
- Text: White, center-aligned, readable

**Example Setup**:
```html

    <!-- Image with responsive srcset -->

              type="image/webp" />
```

## Part 3: Image Sourcing Workflow

### Step 1: Image Research & Selection
**Timeline**: Week 1-2

**Tasks**:
1. Browse Unsplash, Pexels, Pixabay for best matches
2. Create mood board for each category (10-15 images per category)
3. Screenshot or save selected images to reference folder
4. Create spreadsheet tracking:
   - Image source URL
   - License type
   - Photographer name
   - Use case (hero/card/background)
   - Notes on why selected

**Tools**:
- Unsplash: https://unsplash.com
- Pexels: https://www.pexels.com
- Pixabay: https://pixabay.com
- Mood Board Tool: Pinterest or Figma

### Step 2: License Documentation & Acquisition
**Timeline**: Week 2-3

**Tasks**:
1. Document license for each image
2. Download full-resolution versions
3. Store in `/assets/images/original/` with source metadata
4. Create JSON file with license information

**License Metadata Template**:
```json
{
  "images": [
    {
      "id": "wine-hero-001",
      "filename": "wine-elegant-tasting-original.jpg",
      "source": "Unsplash",
      "photographer": "John Smith",
      "sourceUrl": "https://unsplash.com/photos/...",
      "license": "Unsplash License (CC0-like)",
      "commercialUse": true,
      "attributionRequired": false,
      "downloadedDate": "2026-01-15",
      "category": "wine",
      "useCase": ["hero-banner", "featured-section"],
      "originalDimensions": "3840x2160",
      "notes": "Perfect color grading for wine category"
    }
  ]
}
```

### Step 3: Image Optimization & Format Generation
**Timeline**: Week 3

**Process** (for each image):
```bash
# 1. Resize to web dimensions
convert original.jpg -resize 1920x1080 web.jpg

# 2. Convert to WebP (20-30% size reduction)
cwebp web.jpg -o web.webp -q 85

# 3. Create 2x DPI version
convert original.jpg -resize 3840x2160 web-2x.jpg
cwebp web-2x.jpg -o web-2x.webp -q 85

# 4. Create mobile version
convert original.jpg -resize 800x600 mobile.jpg
cwebp mobile.jpg -o mobile.webp -q 80

# 5. Create thumbnail
convert original.jpg -resize 400x300 thumb.jpg

# 6. Optimize all formats
jpegoptim --max=85 *.jpg
```

**Folder Structure**:
```

├── images/
│   ├── original/           # Source files (archive)
│   │   ├── wine/
│   │   ├── coffee/
```

## Part 5: Cost Analysis & Budget

### Free Option (Recommended for Startup)
```
Timeline: 2-3 weeks
Cost: $0

Sources:
- Unsplash (unlimited free downloads)
- Pexels (unlimited free downloads)
- Pixabay (unlimited free downloads)
- Foodiesfeed (unlimited free downloads)

Result: 80+ high-quality images covering all categories

Considerations:
- May need to curate more heavily
- Less exclusivity (images may appear on competitor sites)
- Excellent for MVP and initial launch
```

### Budget Option 1: Adobe Stock Subscription
```
Cost: $9.99/month (10 images/month) or $49.99/month (100 images/month)
Timeline: Ongoing

Recommended: $49.99/month for first 3 months = $150 total

Usage:
- Hero/premium images (10-15)
- Terroir/origin images (5-10)
- Lifestyle photography (10-15)

Fallback to free platforms for product cards

ROI: High quality for key pages, cost-effective
```

### Budget Option 2: Shutterstock Annual Plan
```
Cost: $299/year (100 downloads/month) or $699/year (350 downloads/month)

Recommended: Start with $299/year

Usage:
- All premium category imagery
- Backup for all product photos
- License flexibility for future growth

ROI: Best value if sustained image needs
```

### Hybrid Approach (Recommended for Flavatix)
```
Investment:
- Free platforms: $0 (Unsplash, Pexels, Pixabay)
- Adobe Stock: $49.99/month × 3 months = $150
- Total: $150

Breakdown:
- Free sources: 70% of images (70+ images)
- Adobe Stock: 30% of images (30+ images)

Timeline: 3 weeks
Result: 100+ images covering all categories
Quality: Mix of freely-available and premium
Brand perception: Professional, curated, consistent
```
